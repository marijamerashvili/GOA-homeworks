ჰიპერტექსტის მარკირების ენა (HTML) არის სტანდარტული მარკირების ენა დოკუმენტებისთვის, რომლებიც განკუთვნილია ვებ ბრაუზერში გამოსაჩენად. HTML განსაზღვრავს ვებ კონტენტის შინაარსსა და სტრუქტურას. ის ხშირად გამოიყენება ისეთი ტექნოლოგიების დახმარებით, როგორიცაა კასკადური სტილის ფურცლები (CSS) და სკრიპტირების ენები, როგორიცაა პროგრამირების ენა JavaScript. ვებ ბრაუზერები იღებენ HTML დოკუმენტებს ვებ სერვერიდან ან ადგილობრივი შენახვის საშუალებით და აჩვენებენ მათ მულტიმედიურ ვებ გვერდებზე. HTML აღწერს ვებ გვერდის სტრუქტურას სემანტიკურად და თავდაპირველად შეიცავდა ინფორმაციას მისი გარეგნობის შესახებ. HTML ელემენტები არის HTML გვერდების სამშენებლო ბლოკები. HTML კონსტრუქციების გამოყენებით, გამოსახულებები და სხვა ობიექტები, როგორიცაა ინტერაქტიული ფორმები, შეიძლება ჩაიტვირთოს ჩვენებულ გვერდზე. HTML უზრუნველყოფს სტრუქტურირებულ დოკუმენტებს ისეთი ტექსტური სტრუქტურის საშუალებით, როგორიცაა სათაურები, პუნქტები, სიები, ბმულები, ციტატები და სხვა ელემენტები. HTML ელემენტები განისაზღვრება ტეგებით, რომლებიც წერილობენ კუთხის ფრჩხილებში. ტეგები, როგორიცაა img და input პირდაპირ შემოაქვთ კონტენტი გვერდზე. სხვა ტეგები, როგორიცაა p და /p აყალიბებენ და უზრუნველყოფენ დოკუმენტური ტექსტის შესახებ ინფორმაციას და შეიძლება შეიცავდეს ქვეტეგებს. ბრაუზერები არ აჩვენებენ HTML ტეგებს, არამედ იყენებენ მათ გვერდის შინაარსის გასაგებად. HTML-ში შესაძლებელია პროგრამების ჩასმა სკრიპტირების ენაზე, როგორიცაა JavaScript, რომელიც გავლენას ახდენს ვებ გვერდების ქცევასა და შინაარსზე. CSS-ის ჩართვა განსაზღვრავს კონტენტის ვიზუალსა და განლაგებას. მსოფლიო ქსელის კონსორციუმი (W3C), HTML-ის ყოფილი და CSS სტანდარტების მიმდინარე მმართველი, წაახალისებს CSS-ის გამოყენებას ცნობილი პრეზენტაციული HTML-ის ნაცვლად 1997 წლიდან. HTML-ის ფორმა, ცნობილი როგორც HTML5, გამოიყენება ვიდეო და აუდიოს საჩვენებლად, ძირითადად canvas ელემენტის გამოყენებით, JavaScript-თან ერთად.
